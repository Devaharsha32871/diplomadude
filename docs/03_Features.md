# Key Features

## AI-Powered Assistance
- Answers academic queries using trained AI models.  
- Provides mnemonics, analogies, and examples in responses.  

## Personalized Learning
- Adjusts difficulty level based on user performance.  
- Generates topic suggestions and improvement paths.  

## Video Integration
- Uses YouTube Data API to fetch educational videos per topic.  
- Displays them alongside AI explanations.  

## Secure User Accounts
- Protected login system using Firebase authentication.  
- User data and progress stored securely.  

## Analytics & Feedback
- Tracks user engagement and performance.  
- Allows users to rate explanations for continuous improvement.
